# Flui-figma
O FLUI é um aplicativo mobile desenvolvido com o objetivo de facilitar a localização de pontos de recarga para veículos elétricos, permitindo que usuários encontrem estações próximas de forma rápida e intuitiva.

Com o crescimento da mobilidade elétrica e a ampliação da infraestrutura urbana voltada para sustentabilidade, surgiu a necessidade de criar uma solução que reduzisse barreiras para o uso de carros elétricos, especialmente relacionadas ao planejamento de recarga.

A proposta do aplicativo é centralizar informações importantes em uma interface simples, promovendo melhor experiência de navegação e apoio à mobilidade sustentável.
